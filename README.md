# Buy the Dubai Dip 🇦🇪

Tokenized UAE equity opportunities on Solana.

Highlighting the buying opportunities created by the Iran-UAE conflict — world-class businesses trading at massive discounts to fundamentals.

## Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Deployment:** Vercel
- **Blockchain:** Solana (future integration)

## Development

```bash
npm install
npm run dev
```
